# Diderot Akademi – DILA Testapp

**Encyclopédie 2.0 – Kunskap som befriar**

Intressebaserad AI-pedagogik med:
- **DILM** (Konkret → Representationell → Abstrakt → Reflektion)
- **Curiosity Engine** (XP + nivåer)
- **Dideron** (AI-tutor med demo-svar utan API-nyckel)
- 8 ämnen enligt Skolverket åk 7–9

## Kör lokalt

```bash
npm install
npm run dev
```

Öppna http://localhost:3000

## Design

Mörk renässans-tema (guld på zinc), mobilvänlig chat och ämnesväljare.
